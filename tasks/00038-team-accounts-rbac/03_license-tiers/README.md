# License Tiers & Feature Gates

## Tiers (Simplified - No Workspaces)

| Feature | Free | Pro | Team |
| :--- | :--- | :--- | :--- |
| **Organization** | Personal Only | Personal Only | Team Org |
| **Artifacts** | Unlimited | Unlimited | Unlimited |
| **Version History** | 30 Days | Unlimited | Unlimited |
| **File Size** | 10 MB | 100 MB | 500 MB |
| **Reviewers (Guests)** | 5 per artifact | Unlimited | Unlimited |
| **Team Roles** | ❌ | ❌ | ✅ (Admin/Member) |

## Guest Policy
*   Guests are FREE.
*   Mapped to artifacts via `artifactAccess`, not Org membership.

## Agent Policy
*   Enforced at Org level via `policies.allowAgents`.
