## DIY USPTO Filing Guide

### Step 1: Create USPTO Account
- Go to [USPTO.gov](https://www.uspto.gov/)
- Create account at [MyUSPTO](https://my.uspto.gov/)

### Step 2: Access TEAS (Trademark Electronic Application System)
- Navigate to [TEAS Filing](https://www.uspto.gov/trademarks/apply)
- Select "TEAS Standard" application

### Step 3: Complete Application

**Mark Information:**
- Mark: `Artifact Review`
- Mark type: Standard characters (word mark, no design)

**Owner Information:**
- Entity type and legal name
- Address

**Goods/Services:**
- **Class 42** (primary): SaaS collaboration platform
- Use pre-approved description from [USPTO ID Manual](https://idm-tmng.uspto.gov/id-master-list-public.html) to avoid $200 surcharge
- Draft description: _See "Goods/Services Description" section below_

**Filing Basis:**
- **Intent-to-Use (Section 1b)** - If product not yet launched
- **Use in Commerce (Section 1a)** - If product is live with users

**Specimen (required for Use in Commerce):**
- Screenshot of login page or app interface
- Must show mark clearly associated with the service
- Must show "Login" or service access (not just marketing page)

### Step 4: Pay and Submit
- Fee: **$350/class** (as of Jan 2025)
- Save confirmation and serial number
