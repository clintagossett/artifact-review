# Personas Index

User personas representing target users of the platform.

## Personas

| Persona | Description |
|---------|-------------|
| [Document Creator](./document-creator.md) | PM or knowledge worker generating HTML with AI agents |
| [Document Reviewer](./document-reviewer.md) | Stakeholder reviewing and providing feedback on shared documents |

## Summary

| Persona | Goal | Key Pain Point |
|---------|------|----------------|
| **Document Creator** | Share AI artifacts for review | Format loss when converting to Google Docs |
| **Document Reviewer** | Provide inline feedback | Can't comment on HTML artifacts directly |

## Flow

1. Document Creator uploads HTML artifact
2. Invites Document Reviewers
3. Reviewers provide feedback
4. Creator iterates based on comments
