# Document Reviewer

**Role:** Stakeholder, teammate, or approver reviewing shared documents

## Context

- May not use AI agents themselves
- Invited via link to review specific documents
- Needs to provide feedback without learning new tools

## Goals

1. View documents exactly as intended
2. Leave feedback on specific sections
3. Track what needs attention

## Pain Points

- Receiving files in awkward formats
- No easy way to comment inline on HTML
- Losing context across email/Slack threads

## Key Actions

- Open shared document links
- Read and navigate documents
- Highlight text and add comments
- @mention others for input
- Mark feedback as resolved
