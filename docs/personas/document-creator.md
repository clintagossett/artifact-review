# Document Creator

**Role:** PM, knowledge worker, or anyone generating HTML artifacts with AI agents

## Context

- Uses Claude Code, Cursor, or similar AI coding agents
- Generates HTML documents: PRDs, specs, reports, presentations, prototypes
- Needs stakeholder feedback before shipping

## Goals

1. Share AI-generated HTML without format conversion
2. Get feedback from reviewers quickly
3. Track review status and approvals

## Pain Points

- Converting HTML to Google Docs loses formatting
- Sharing raw files is clunky, no collaboration
- Screenshots aren't interactive

## Key Actions

- Upload HTML documents
- Generate shareable links
- Invite reviewers
- Respond to comments
- Upload revised versions
