# Complex Markdown Demo Sample

This sample contains complex markdown files used for testing advanced rendering features.

## Contents

- `v1.md`: Features Mermaid diagrams (Flowchart, Sequence, Gantt), GFM tables, task lists, and code blocks.

## Purpose

To verify that the markdown renderer correctly handles:
1. Mermaid diagrams (Phase 2)
2. GFM extensions (Tables, Task lists)
3. Code syntax highlighting
4. Standard markdown typography
