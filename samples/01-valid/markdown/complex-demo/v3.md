# Complex Markdown Demo - Version 3

This document demonstrates complex markdown features, including GitHub-flavored markdown (GFM) and Mermaid diagrams.

## 1. Mermaid Diagrams

### New: Class Diagram
```mermaid
classDiagram
    Animal <|-- Duck
    Animal <|-- Fish
    Animal <|-- Zebra
    Animal : +int age
    Animal : +String gender
    Animal: +isMammal()
    Animal: +mate()
    class Duck{
        +String beakColor
        +swim()
        +quack()
    }
    class Fish{
        -int sizeInFeet
        -canEat()
    }
    class Zebra{
        +bool is_wild
        +run()
    }
```
