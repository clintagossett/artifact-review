# Feature-Rich Markdown Sample

Single-file markdown with all markdown features for comprehensive annotation testing.

## Features

| Category | Elements |
|----------|----------|
| Text | Bold, italic, strikethrough, inline code |
| Headings | h1 through h6 |
| Lists | Ordered, unordered, nested, task lists |
| Code | JS, TS, Python, SQL, Bash, JSON, YAML, Go, Rust |
| Tables | Simple and complex with alignment |
| Quotes | Simple, nested, with formatting |
| Diagrams | Mermaid flowcharts and sequence diagrams |
| Links | External, anchor links, images |
| Other | Horizontal rules, long paragraphs |

## Testing Scenarios

1. **Text Selection** - Select within paragraphs, across formatting
2. **Code Annotation** - Select within code blocks
3. **Table Annotation** - Select table cells
4. **Anchor Navigation** - Test TOC links
5. **Long Content** - Test scroll + annotation persistence

## Usage

Upload `v1.md` directly as a single-file markdown artifact.
